# dotfiles
This repository is my private dotfiles(But anyone can use this).

# Setup
## Install vim and gvim
~~~~
$ sudo apt-get update
$ sudo apt-get install vim
$ sudo apt-get install vim-gnome
~~~~

## Clone the dotfiles

~~~~
$ git clone https://gist.github.com/e4243dc05c506bbd92c3.git tmp
$ mv tmp/vimrc  .vimrc
~~~~
